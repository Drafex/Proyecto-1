# Proyecto-1
Instalar: 
1)instale node js
2)instale sails.js
3)instale git en caso de que no esté
4)clone el repositorio con: https://github.com/Drafex/Proyecto-1.git o ingrese a https://tranquil-mountain-21660.herokuapp.com
5)si decidió clonar lo en la carpeta donde esta el folder telemática ingrese el comando sails lift
6)diríjase al browser e ingrese al link http://localhost:1337/
7)y siga las instrucciones, en caso de error revise el path
8)si decidió ir por el link siga las instrucciones que le da la app
9)para visualizar la creacion de usuarios dirigase a https://tranquil-mountain-21660.herokuapp.com/user/
10)gracias
